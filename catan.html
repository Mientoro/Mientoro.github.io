Electronic Catan Board

As an avid board game fan, I decided to make a fancy version of the classic Settlers of Catan game. Using indivually addressable LEDs, an Arduino board, and some nifty wiring, I am in the process of making a unique game board.

The board has several features. The LEDs allow for multiple lighting effects in any color. As a result, I created some custom 3D printed resource tiles that were shelled to let light shine through. This lets the LEDs light each resource tile a specific color to reflect that type of resource. Each tile has connectors in it that can be connected at ports in every tile location that route back to the Arduino board. The board can use these connections to determine which resource is at which location. In addition, simple light sequences are coded into the Arduino to reflect different events in the game. The Arduino board uses a code form of dice to roll on the click of a button and can then highlight corresponding tiles in the game.

A future addition to the board will be dice embedded with RFID tags which can be detected to determine what number the dice are rolled on. This is read by the Arduino to replace the coded dice rolls with real ones.
